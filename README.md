## Usage

Для ознакомления с проектом и примерами использования запустите следующие ноутбуки:

- **example_usage.ipynb** — базовый пример использования основных компонентов
- **example_usage_trellis.ipynb** — пример использования с интеграцией Trellis

Оба ноутбука содержат пошаговые демонстрации и предназначены для запуска в Jupyter Notebook / Jupyter Lab.
